## 2. Тест-план ручного тестирования

### Условия тестирования

| Параметр | Значение |
|----------|---------|
| ОС | Windows 10/11 |
| JDK | 17+ |
| Разрешение экрана | 1920×1080 (минимум 1500×1000) |
| База данных | SQLite (файловая, создаётся автоматически) |
| Состояние БД перед тестами | Чистая (первый запуск) или с тестовыми данными |

### Уровни приоритета

| Приоритет | Описание |
|-----------|---------|
| 🔴 **P0** | Критический — блокирующий, приложение неработоспособно |
| 🟠 **P1** | Высокий — ключевая функция нарушена |
| 🟡 **P2** | Средний — некритичный дефект |
| 🟢 **P3** | Низкий — косметика, удобство |
