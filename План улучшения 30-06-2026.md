## 📊 Выполненные оптимизации (БД)

- [x] **Пул соединений** — HikariCP (вместо DriverManager)
- [x] **Индексы** — для всех внешних ключей и часто фильтруемых полей
- [x] **JOIN-запросы** — устранён N+1 в `getAllOrders()`
- [x] **Транзакции** — для `addOrder()` и `updateOrder()`
- [x] **Инкрементальное обновление** — DataStore без полной перезагрузки
- [x] **Оптимизация генерации ID** — MAX вместо ORDER BY

## 📎 Примечания

- Все улучшения должны быть **обратно совместимы** с текущей версией БД
- Внедрять изменения **постепенно**, тестируя после каждого этапа
- Код должен соответствовать **Java 17** стилю
- UI должен быть написан на **JavaFX + CSS** (без SceneBuilder, чтобы сохранить контроль над кодом)
