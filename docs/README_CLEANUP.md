# Скрипты очистки связей услуга-запчасть

## Описание

В директории `docs/` созданы два скрипта для очистки связей между услугами и запчастями:

### 1. `cleanup_service_spare_parts.py` - Полная очистка

Удаляет **ВСЕ** связи между услугами и запчастями:
- service_spare_parts (старая структура)
- service_spare_parts_lists (новая структура)
- service_spare_parts_list_items (элементы списков)

**Использование:**
```bash
# Из корневой директории проекта
python docs/cleanup_service_spare_parts.py

# Или с указанием пути к БД
python docs/cleanup_service_spare_parts.py autoservice.db
```

### 2. `cleanup_duplicates.py` - Удаление только дубликатов

Удаляет только дубликаты, оставляя по одной связи для каждой пары услуга-запчасть.

**Использование:**
```bash
# Из корневой директории проекта
python docs/cleanup_duplicates.py

# Или с указанием пути к БД
python docs/cleanup_duplicates.py autoservice.db
```

## Предупреждение

⚠️ Перед запуском скриптов **обязательно создайте резервную копию базы данных**:

```bash
# Простой способ - скопировать файл вручную
cp autoservice.db autoservice.db.backup

# Или создайте бэкап через настройки приложения
```

## Проверка результатов

После выполнения скрипта вы увидите отчет о количестве удаленных записей и проверку результата.

## Альтернативный способ через интерфейс

В приложении также есть кнопка **"Очистить дубликаты"** в настройках таблицы "Связи услуг-запчастей". Она использует ту же логику, что и скрипт `cleanup_duplicates.py`.
