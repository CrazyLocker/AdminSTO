# Спецификация: Сохранение состояния TableView в JSON для JavaFX

## Версия: 1.0
## Дата: 2026-07-12

---

## 1. Цель

Разработать механизм сохранения и восстановления пользовательских настроек таблицы `javafx.scene.control.TableView` между сессиями приложения.

**Ключевые требования:**
- Сохранение состояния в JSON-формат
- Полное восстановление пользовательских настроек при следующем запуске
- Отсутствие изменений в архитектуре JavaFX
- Поддержка нескольких таблиц в одном приложении
- Работа в портативном режиме (с флешки) при необходимости

---

## 2. Сохраняемые параметры таблицы

| № | Параметр | Описание | Источник данных в JavaFX | Тип данных |
|---|----------|----------|--------------------------|------------|
| 1 | **Ширина колонок** | Текущая ширина каждой колонки в пикселях | `TableColumn.getWidth()` | `double` |
| 2 | **Порядок колонок** | Порядок следования колонок после перетаскивания пользователем | `TableView.getColumns()` (порядок в списке) | `List<TableColumn>` |
| 3 | **Видимость колонок** | Состояние видимости каждой колонки (показана/скрыта) | `TableColumn.isVisible()` | `boolean` |
| 4 | **Сортировка** | Колонка и направление сортировки (по возрастанию/убыванию) | `TableView.getSortOrder()` | `List<TableColumn>` + `SortOrder` |

---

## 3. Структура JSON-файла

```json
{
  "tableId": "customersTable",
  "version": "1.0",
  "timestamp": "2026-07-12T07:45:39Z",
  "columns": [
    {
      "id": "name",
      "width": 150.0,
      "visible": true,
      "index": 0
    },
    {
      "id": "email",
      "width": 250.0,
      "visible": true,
      "index": 1
    },
    {
      "id": "phone",
      "width": 120.0,
      "visible": false,
      "index": 2
    },
    {
      "id": "address",
      "width": 300.0,
      "visible": true,
      "index": 3
    }
  ],
  "sortOrder": [
    {
      "columnId": "name",
      "order": "ASCENDING"
    }
  ]
}
