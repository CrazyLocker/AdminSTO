# План: Создание новой вкладки в главном окне

**Дата создания плана:** 12.07.2026  
**Задача:** Создать вкладку в главном окне с таблицей 5 колонок и 10 строк, оформленную как у Клиентов.

## Выполненные действия

### 1. Создан файл NewView.java
**Расположение:** `src/main/java/com/autoservice/views/NewView.java`

**Описание:**
- Создана новая вкладка с оформлением, идентичным вкладке "Клиенты"
- Таблица с 5 колонками (Колонка 1-5)
- Автоматически заполняется 10 строками данных при первом запуске
- Есть кнопки: Добавить, Изменить, Удалить
- Поддерживается двойной клик для редактирования
- Используется тот же стиль таблицы (table-view) и CSS-классы

**Структура:**
- Метод `create()` — создание всей вкладки
- Метод `createNewTable()` — настройка таблицы с 5 колонками
- Метод `refreshList()` — обновление данных (заполнение 10 строками)
- Класс `NewRow` — модель данных строки таблицы

### 2. Обновлен файл App.java
**Расположение:** `src/main/java/com/autoservice/App.java`

**Изменения:**
- Добавлен импорт для новой вкладки (уже есть через `import com.autoservice.views.*;`)
- Создана новая вкладка: `Tab newTab = createTab("Новая вкладка", IconHelper.report());`
- Добавлена содержимое: `newTab.setContent(NewView.create());`
- Добавлена вкладка в tabPane: `tabPane.getTabs().addAll(dashTab, clientTab, orderTab, dictTab, appointmentTab, newTab);`
- Исправлено дублирование переменной `newTab` в listener'е

### 3. Проверка сборки
Команда: `mvn clean compile -DskipTests`  
Статус: **BUILD SUCCESS**

## Оформление (в соответствии с ClientView)

### Таблица
- Стиль: `-fx-background-color: transparent`
- Граница: `-fx-border-color: #e0e0e0`
- Скругление: 12px
- Заголовок колонок: темно-синий фон `#381CC1`, белый текст
- Высота строки: 37px
- Чередование строк: белый/светло-серый
- Выделение: светло-синий при выборе

### Кнопки
- Новый/Добавить: зеленый (`-success-color`)
- Изменить: оранжевый (`-warning-color`)
- Удалить: красный (`-danger-color`)
