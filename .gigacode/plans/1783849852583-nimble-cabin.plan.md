# План реализации: Сохранение состояния TableView

**Версия:** 1.0  
**Дата:** 2026-07-12

---

## 1. Обзор

Реализовать механизм сохранения и автоматического восстановления пользовательских настроек TableView (ширина колонок, порядок, видимость, сортировка) в JSON-файлах.

---

## 2. Архитектура решения

### 2.1. Структура данных

**Формат JSON:**
```json
{
  "tableId": "orders",
  "version": "1.0",
  "timestamp": "2026-07-12T10:30:00Z",
  "columns": [
    {"id": "id", "width": 50.0, "visible": true, "index": 0},
    {"id": "date", "width": 120.0, "visible": true, "index": 1},
    {"id": "amount", "width": 100.0, "visible": true, "index": 2},
    {"id": "status", "width": 80.0, "visible": false, "index": 3}
  ],
  "sortOrder": [
    {"columnId": "date", "order": "DESCENDING"}
  ]
}
```

**Пример сохраненного состояния (orders):**
- 4 колонки: id, date, amount, status
- Колонка status скрыта
- Сортировка по дате по убыванию
- Ширины колонок: 50, 120, 100, 80 пикселей соответственно

**Хранение:** `config/table-state/{tableId}.json`

### 2.2. Классы

| Класс | Назначение |
|-------|------------|
| `TableState.java` | Модель данных для JSON |
| `TableStateManager.java` | Основной API для управления состоянием |
| `TableViewUtils.java` | Утилиты для работы с TableView |

### Шаг 3: Полную реализацию `TableStateManager.java`

**Файл:** `src/main/java/com/autoservice/services/TableStateManager.java`

**Полная реализация (адаптированная под проект):**

```java
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.databind.SerializationFeature;
import com.fasterxml.jackson.datatype.jsr310.JavaTimeModule;
import javafx.collections.ObservableList;
import javafx.scene.control.TableColumn;
import javafx.scene.control.TableView;
import javafx.scene.control.TableColumn.SortType;

import java.io.File;
import java.io.IOException;
import java.time.Instant;
import java.util.*;
import java.util.stream.Collectors;

public class TableStateManager {
    
    private static final String STORAGE_DIR = "config/table-state";
    private static final ObjectMapper mapper = new ObjectMapper()
            .registerModule(new JavaTimeModule())
            .enable(SerializationFeature.INDENT_OUTPUT);
    private static final double DEFAULT_COLUMN_WIDTH = 100.0;
    
    // ========== ОСНОВНЫЕ МЕТОДЫ API ==========
    
    public static void saveTableState(TableView<?> table, String tableId) {
        if (table == null || tableId == null || tableId.trim().isEmpty()) return;
        
        try {
            TableState state = new TableState();
            state.setTableId(tableId);
            state.setVersion("1.0");
            state.setTimestamp(Instant.now().toString());
            state.setColumns(extractColumnStates(table));
            state.setSortOrder(extractSortStates(table));
            
            File file = getStateFile(tableId);
            mapper.writeValue(file, state);
            
        } catch (IOException e) {
            System.err.println("Ошибка сохранения состояния таблицы: " + e.getMessage());
        }
    }
    
    public static void loadTableState(TableView<?> table, String tableId) {
        if (table == null || tableId == null || tableId.trim().isEmpty()) return;
        
        File file = getStateFile(tableId);
        if (!file.exists()) return;
        
        try {
            TableState state = mapper.readValue(file, TableState.class);
            if (state == null || state.getColumns() == null) return;
            
            // Восстанавливаем ширину и видимость колонок
            restoreColumnStates(table, state.getColumns());
            
            // Восстанавливаем порядок колонок (если есть index)
            restoreColumnOrder(table, state.getColumns());
            
            // Добавляем отсутствующие колонки из JSON
            addMissingColumns(table, state.getColumns());
            
            // Восстанавливаем сортировку
            restoreSortOrder(table, state.getSortOrder());
            
        } catch (IOException e) {
            System.err.println("Ошибка загрузки состояния таблицы: " + e.getMessage());
        }
    }
    
    public static void resetTableState(String tableId) {
        if (tableId == null || tableId.trim().isEmpty()) return;
        
        File file = getStateFile(tableId);
        if (file.exists()) file.delete();
    }
    
    public static boolean hasSavedState(String tableId) {
        if (tableId == null || tableId.trim().isEmpty()) return false;
        return getStateFile(tableId).exists();
    }
    
    // ========== ВСПОМОГАТЕЛЬНЫЕ МЕТОДЫ ==========
    
    private static File getStateFile(String tableId) {
        File dir = new File(STORAGE_DIR);
        if (!dir.exists()) dir.mkdirs();
        return new File(dir, tableId + ".json");
    }
    
    private static List<ColumnState> extractColumnStates(TableView<?> table) {
        List<ColumnState> states = new ArrayList<>();
        ObservableList<? extends TableColumn<?, ?>> columns = table.getColumns();
        
        for (int i = 0; i < columns.size(); i++) {
            TableColumn<?, ?> column = columns.get(i);
            ColumnState state = new ColumnState();
            state.setId(column.getId() != null ? column.getId() : "col_" + i);
            state.setWidth(column.getWidth());
            state.setVisible(column.isVisible());
            state.setIndex(i);
            states.add(state);
        }
        return states;
    }
    
    private static List<SortState> extractSortStates(TableView<?> table) {
        List<SortState> states = new ArrayList<>();
        ObservableList<TableColumn<?, ?>> sortOrder = table.getSortOrder();
        
        for (TableColumn<?, ?> column : sortOrder) {
            SortState state = new SortState();
            state.setColumnId(column.getId() != null ? column.getId() : "col_" + column.getText());
            state.setOrder(column.getSortType() == SortType.ASCENDING ? "ASCENDING" : "DESCENDING");
            states.add(state);
        }
        return states;
    }
    
    private static void restoreColumnStates(TableView<?> table, List<ColumnState> columnStates) {
        ObservableList<? extends TableColumn<?, ?>> columns = table.getColumns();
        
        for (ColumnState cs : columnStates) {
            if (cs.getId() == null) continue;
            
            TableColumn<?, ?> foundColumn = columns.stream()
                    .filter(col -> cs.getId().equals(col.getId()))
                    .findFirst()
                    .orElse(null);
            
            if (foundColumn != null) {
                foundColumn.setPrefWidth(cs.getWidth());
                foundColumn.setVisible(cs.isVisible());
            }
        }
    }
    
    private static void restoreColumnOrder(TableView<?> table, List<ColumnState> columnStates) {
        // Собираем существующие колонки в список для удобства
        List<TableColumn<?, ?>> existingColumns = new ArrayList<>(table.getColumns());
        List<TableColumn<?, ?>> orderedColumns = new ArrayList<>();
        Set<String> processedIds = new HashSet<>();
        
        // Сортируем сохранённые состояния по индексу
        List<ColumnState> sortedStates = columnStates.stream()
                .filter(cs -> cs.getIndex() != null)
                .sorted((a, b) -> Integer.compare(a.getIndex(), b.getIndex()))
                .collect(Collectors.toList());
        
        // Сначала добавляем колонки, которые есть и в JSON, и в таблице, в правильном порядке
        for (ColumnState cs : sortedStates) {
            TableColumn<?, ?> foundColumn = existingColumns.stream()
                    .filter(col -> cs.getId() != null && cs.getId().equals(col.getId()))
                    .findFirst()
                    .orElse(null);
            
            if (foundColumn != null) {
                orderedColumns.add(foundColumn);
                processedIds.add(cs.getId());
            }
        }
        
        // Добавляем оставшиеся колонки из таблицы (в их текущем порядке)
        for (TableColumn<?, ?> col : existingColumns) {
            if (!processedIds.contains(col.getId())) {
                orderedColumns.add(col);
            }
        }
        
        // Если порядок изменился, обновляем таблицу
        if (orderedColumns.size() != existingColumns.size()) return;
        
        table.getColumns().setAll(orderedColumns);
    }
    
    private static void addMissingColumns(TableView<?> table, List<ColumnState> columnStates) {
        Set<String> existingIds = table.getColumns().stream()
                .map(col -> col.getId() != null ? col.getId() : "col_" + table.getColumns().indexOf(col))
                .collect(Collectors.toSet());
        
        List<ColumnState> missingStates = columnStates.stream()
                .filter(cs -> !existingIds.contains(cs.getId()))
                .collect(Collectors.toList());
        
        for (ColumnState cs : missingStates) {
            TableColumn<Object, Object> newCol = new TableColumn<>(cs.getId());
            newCol.setId(cs.getId());
            newCol.setPrefWidth(cs.getWidth() > 0 ? cs.getWidth() : DEFAULT_COLUMN_WIDTH);
            newCol.setVisible(cs.isVisible());
            table.getColumns().add(newCol);
        }
    }
    
    private static void restoreSortOrder(TableView<?> table, List<SortState> sortStates) {
        if (sortStates == null || sortStates.isEmpty()) {
            table.getSortOrder().clear();
            return;
        }
        
        table.getSortOrder().clear();
        
        for (SortState ss : sortStates) {
            if (ss.getColumnId() == null || ss.getOrder() == null) continue;
            
            TableColumn<?, ?> column = table.getColumns().stream()
                    .filter(col -> ss.getColumnId().equals(col.getId()))
                    .findFirst()
                    .orElse(null);
            
            if (column != null) {
                SortType sortType = "DESCENDING".equals(ss.getOrder()) 
                        ? SortType.DESCENDING 
                        : SortType.ASCENDING;
                column.setSortType(sortType);
                table.getSortOrder().add(column);
            }
        }
    }
}
```

**Ключевые изменения под проект:**
1. Путь: `config/table-state/` (относительный)
2. Добавление отсутствующих колонок в конец с шириной 100px
3. Использование Jackson для JSON

### Шаг 4: Добавление слушателей для автоматического сохранения

**После создания таблицы добавить слушатели:**
```java
table.getColumns().addListener((Change<?> c) -> {
    TableStateManager.saveTableState(table, table.getId());
});

table.getSortOrder().addListener((Change<?> c) -> {
    TableStateManager.saveTableState(table, table.getId());
});

for (TableColumn<?, ?> col : table.getColumns()) {
    col.widthProperty().addListener((obs, oldVal, newVal) -> {
        TableStateManager.saveTableState(table, table.getId());
    });
    col.visibleProperty().addListener((obs, oldVal, newVal) -> {
        TableStateManager.saveTableState(table, table.getId());
    });
}
```

## 4. Управление колонками

### 4.1. Назначение ID колонкам

**Для каждой колонки установить ID:**
```java
colLastName.setId("colLastName");
colName.setId("colName");
```

**Если ID не задан, использовать заголовок:**
```java
String columnId = column.getId() != null ? column.getId() : 
                  generateColumnId(column.getText());
```

### 4.2. Порядок колонок

**Восстановление порядка:**
- Колонки сортируются по полю `index` из JSON
- Колонки, отсутствующие в JSON, сохраняют свой относительный порядок в конце
- Используется `table.getColumns().setAll(orderedColumns)` для массового обновления

---

## 5. Обработка отсутствующих колонок

**Алгоритм:**
1. При восстановлении сравнить ID из JSON с существующими колонками
2. Сначала применить настройки к существующим колонкам
3. Добавить отсутствующие колонки из JSON в конец таблицы
4. Колонки без ID получают дефолтную ширину (100px)

## 6. Обновление состояния

**При изменении таблицы автоматически обновлять JSON:**

| Событие | Действие |
|---------|----------|
| Изменение ширины | `saveTableState()` |
| Перетаскивание колонок | `saveTableState()` |
| Показ/скрытие колонки | `saveTableState()` |
| Изменение сортировки | `saveTableState()` |

**Пример слушателя:**
```java
table.getColumns().addListener((Change<?> c) -> {
    TableStateManager.saveTableState(table, table.getId());
});
```
