# План изменения вкладки "Расходники ТО"

## Общее описание
Изменить вкладку "Расходники ТО" (ToParts) чтобы она работала по аналогии с вкладкой "Связь услуги-запчасти", но с моделями авто вместо услуг.

## Текущее состояние

### Вкладка "Расходники ТО" (SettingsView.java:681-844)
- Имеет форму добавления **под таблицей** (не по шаблону)
- Поля: Модель авто (TextField), Запчасть (ComboBox), Кол-во, Ед. изм.
- Кнопка "Добавить связь"
- Нет кнопок редактирования и удаления

### Вкладка "Связь услуги-запчасти" (SettingsView.java:530-660) - эталон
- Кнопки управления **сверху**: Добавить, Изменить, Удалить, Обновить
- Таблица: Услуга, Запчасти (список через запятую)
- Диалог добавления с множественным выбором запчастей

## Изменения для вкладки "Расходники ТО"

### 1. Перенос кнопок управления наверх
```java
// Верхняя панель с кнопками (как в createServiceSparePartsPanel)
HBox topButtons = new HBox(10);
topButtons.setAlignment(Pos.CENTER_LEFT);

Button addBtn = new Button("Добавить");
Button editBtn = new Button("Изменить");
Button deleteBtn = new Button("Удалить");
Button refreshBtn = new Button("Обновить список");

topButtons.getChildren().addAll(addBtn, editBtn, deleteBtn, refreshBtn);
```

### 2. Удаление формы добавления под таблицей
Удалить HBox formRow с полями carModelField, sparePartCombo и т.д.

### 3. Изменение метода добавления
Переименовать кнопку "Добавить связь" → "Добавить"

По клике открывать диалог аналогичный `showAddServiceSparePartDialog()` но с отличиями:
- **Выпадающий список моделей авто** вместо услуг
- Таблица запчастей с множественным выбором (как в dialog)
- Добавить поле **Примечание** (TextArea на 1000 символов)

### 4. Добавление кнопки "Удалить"
```java
deleteBtn.setOnAction(e -> {
    ToPartRow selected = toPartsTable.getSelectionModel().getSelectedItem();
    if (selected != null) {
        DataStore.deleteToPart(selected.getToPart());
        refreshToParts();
    }
});
```

### 5. Добавление редактирования
Добавить кнопку "Изменить" и метод `showEditToPartDialog()`

### 6. Создание ToPartsRow класса
Создать модель отображения аналог `ServiceSparePartsRow` для таблицы:
```java
public class ToPartsRow {
    private ToPart toPart;
    private String carModel;
    private String sparePartName;
    private int quantity;
    private String unitType;
    private String note;
    // getters/setters
}
```

### 7. Обновление таблицы
Добавить колонку "Примечание" (показывать первые 50 символов с "..." и tooltip для полного текста)

### 8. Обновление модели ToPart
Добавить поле `note` (VARCHAR(1000)) и геттеры/сеттеры

### 9. Обновление БД
Добавить колонку `note` в таблицу `to_parts`:
```sql
ALTER TABLE to_parts ADD COLUMN note VARCHAR(1000);
```

### 10. Обновление DataStore и SettingsController
Добавить методы обновления/удаления для новой структуры
