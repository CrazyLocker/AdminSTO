## Приоритет 1: Исправление критичных багов

### 1.1 Исправить импорт с RETURN_GENERATED_KEYS
**Файл:** `src/main/java/com/autoservice/Database.java`
**Строка:** 749

**Проблема:** SQLite JDBC driver не поддерживает `Statement.RETURN_GENERATED_KEYS` для `INSERT`.

**Решение:** Получать ID записи через `SELECT LAST_INSERT_ROWID()` после `INSERT`.

```java
// Текущий код (не работает):
PreparedStatement pstmt = conn.prepareStatement(sql, Statement.RETURN_GENERATED_KEYS);

// Исправленный код:
String sql = "INSERT INTO appointments (client_id, order_id, master_name, service_name, appointment_date, appointment_time, status) VALUES (?, ?, ?, ?, ?, ?, ?)";
try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
    // ... установка параметров
    pstmt.executeUpdate();
    
    // Получить ID через LAST_INSERT_ROWID()
    try (Statement stmt = conn.createStatement();
         ResultSet rs = stmt.executeQuery("SELECT LAST_INSERT_ROWID()")) {
        if (rs.next()) {
            appointment.setId(rs.getInt(1));
        }
    }
}
```

**Файлы, которые нужно исправить:**
- `Database.addAppointment()` - получение ID записи
- `Database.updateAppointment()` - обновление записи
- `Database.deleteAppointment()` - удаление записи

```xml
<plugin>

    <version>0.8.11</version>
    <executions>
        <execution>
            <goals>
                <goal>prepare-agent</goal>
            </goals>
        </execution>
        <execution>
            <id>report</id>
            <phase>test</phase>
            <goals>
                <goal>report</goal>
            </goals>
        </execution>
    </executions>
</plugin>
```

## Проверка выполнения

После выполнения всех изменений:

```bash
# Запуск всех тестов
mvn test

# Генерация отчёта JaCoCo
mvn test jacoco:report

# Запуск только unit-тестов
mvn test -Dgroups=unit

# Запуск только интеграционных тестов
mvn test -Dgroups=integration
```

**Критерии успеха:**
- Все тесты проходят (`mvn test` без ошибок)
- Покрытие кода ≥85%
- Время выполнения <10 сек
- 0 flaky тестов
