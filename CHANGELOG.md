# Changelog

Все значимые изменения проекта `autoservice-admin`.

## [0.0.9] - 2024-12

### Добавлено
- **Импорт запчастей из файлов** (CSV, XML, JSON)
  - Поддержка формата CSV с разделителем `;`
  - Поддержка XML формата `<spareParts><part>...</part></spareParts>`
  - Поддержка JSON формата `{"spareParts": [...]}` и массивов
  - Отчёт об ошибках импорта
  - Сохранение последнего пути к файлу импорта (Preferences API)
  
- **Вкладка "Склад"**
  - Просмотр всех запчастей с остатками
  - Установка минимального остатка для каждой запчасти
  - Внесение прихода запчастей
  - Контроль запчастей с остатком ниже минимального
  
- **Редактирование запчастей**
  - Двойной клик на вкладке "Запчасти" → редактирование всех полей
  - Двойной клик на вкладке "Склад" → редактирование мин. остатка и прихода
  
- **Массовое удаление запчастей**
  - Выделение нескольких строк (Ctrl+Click, Shift+Click)
  - Подтверждение удаления с указанием количества
  
- **Поиск запчастей**
  - Фильтрация по названию, артикулу, производителю

### Изменено
- Вкладка "Приход запчастей" переименована в "Склад"
- Кнопка "Импорт" перемещена на вкладку "Запчасти"
- Кнопка "Внести приход" на складе активна только при выбранной строке

### Исправлено
- **Обновление таблицы после импорта** — таблицы обновляются сразу после импорта без перезапуска
- **Сохранение минимального остатка** — значение min_stock теперь сохраняется в БД
- **UTF-8 кодировка** — все файлы проекта в UTF-8, настроен компилятор

### Технические изменения
- Добавлены тесты `ImportServiceTest` (15 тестов)
- Добавлены тесты `ImportSparePartsDialogTest` (10 тестов)
- Обновлена матрица трассируемости требований
- Версия POM обновлена до `0.0.9`

## [0.0.8] - 2024-11

### Добавлено
- Дашборд с ключевыми метриками

## Формат

Проект использует семантическое версионирование [MAJOR.MINOR.PATCH](https://semver.org/lang/ru/):
- **MAJOR** — крупные несовместимые изменения
- **MINOR** — новая функциональность
- **PATCH** — исправления ошибок
