# План развития проекта "Администратор СТО"

## Текущий статус (v1.0)
- ✅ Управление клиентами (CRUD, поиск)
- ✅ Управление заказами (CRUD, статусы, поиск)
- ✅ Справочники (услуги, запчасти, склад)
- ✅ Календарь записи (день/неделя/месяц)
- ✅ SQLite база данных
- ✅ Валидация телефона и госномера
- ✅ Flat Design
- ✅ Поиск по клиентам и заказам
- ✅ Автотесты (JUnit 5 + AssertJ)

## План развития

### 1. Финансовый блок 💰
- [ ] Учёт доходов/расходов (касса)
- [ ] Зарплата мастеров (процент от выработки)
- [ ] Отчёт по прибыли за период
- [ ] Накладные расходы (аренда, коммуналка)
- [ ] Финансовый дашборд

### 2. Печать документов 🖨️
- [ ] Заказ-наряд (PDF с логотипом СТО)
- [ ] Акт выполненных работ
- [ ] Чек (фискальный/нефискальный)
- [ ] Печать нескольких заказов сразу

### 3. Экспорт в Excel/CSV 📊
- [ ] Выгрузка таблиц (клиенты, заказы, запчасти)
- [ ] Финансовые отчёты
- [ ] Оборотно-сальдовая ведомость
- [ ] Отчёт по клиентам (частота, средний чек)

### 4. Уведомления 🔔
- [ ] SMS/Email клиенту о готовности авто
- [ ] Напоминание о записи (за день)
- [ ] Уведомление о низком остатке запчастей
- [ ] Настройка шаблонов уведомлений

### 5. Статистика и графики 📈
- [ ] График выручки по дням/месяцам
- [ ] Топ услуг и запчастей
- [ ] Загрузка мастеров (диаграмма)
- [ ] Аналитика по клиентам
- [ ] Прогнозирование загрузки

### 6. Интеграция с веб-сайтом 🌐
- [ ] REST API для онлайн-записи
- [ ] Импорт заявок с сайта
- [ ] Экспорт статусов заказов
- [ ] Webhook-уведомления

### 7. Резервное копирование 💾
- [ ] Автоматический бэкап базы данных
- [ ] Восстановление из бэкапа
- [ ] Экспорт/импорт всех данных (JSON/XML)
- [ ] Планировщик бэкапов

### 8. Улучшение существующего 🔧
- [ ] Тёмная тема
- [ ] Горячие клавиши
- [ ] Подтверждение действий (undo/redo)
- [ ] Валидация всех полей ввода
- [ ] Drag-and-drop для календаря
- [ ] Поиск с фильтрами

### 9. Многопользовательский режим 👥
- [ ] Авторизация (логин/пароль)
- [ ] Роли (администратор, мастер, бухгалтер)
- [ ] Разные права доступа
- [ ] Логирование действий

### 10. Мобильное приложение 📱
- [ ] Просмотр расписания
- [ ] Уведомления
- [ ] Быстрый доступ к клиентам
