# AdminSTO v1.0.0 - Система управления автосервисом

## 📦 Описание

AdminSTO — настольное приложение для автоматизации рабочих процессов автосервиса на базе JavaFX.

## 🚀 Портативная версия

Для удобства запуска без предустановленной Java доступна **портативная версия**:

1. Распакуйте `AdminSTO_Portable/AdminSTO_Portable.zip` в любую папку
2. Запустите `AdminSTO_Portable/STO.bat`

Портативная версия включает встроенный JRE 21 и готова к работе сразу после распаковки.

## ✨ Возможности

- **Управление клиентами** — база данных клиентов с информацией об автомобилях
- **Заказы-наряды** — создание и ведение заказов с услугами и запчастями
- **Экспорт заказа в PDF** — сохранение заказа-наряда в формате PDF
- **Справочники** — услуги и запчасти с ценами
- **Записи на обслуживание** — календарь записей клиентов
- **Резервное копирование** — автоматические бэкапы базы данных
- **Отчёты** — генерация PDF-отчётов (общая статистика, динамика выручки)
- **Импорт** — загрузка запчастей из CSV файлов
- **Экспорт данных** — экспорт клиентов, услуг и запчастей в CSV/XML/JSON

## 🔧 Системные требования

### Для сборки из исходников:
- **ОС:** Windows 10/11, Linux, macOS
- **Java:** JDK 21 или выше
- **ОЗУ:** Минимум 512 МБ
- **Место на диске:** Минимум 1 ГБ (с Gradle кэшем)

### Для запуска готовых JAR:
- **ОС:** Windows 10/11, Linux, macOS
- **Java:** JRE 21 или выше
- **ОЗУ:** Минимум 512 МБ
- **Место на диске:** Минимум 100 МБ

### Портативная версия:
- **ОС:** Windows 10/11 (текущая версия)
- **Место на диске:** ~102 МБ
- **Java:** Встроенный JRE 21 (не требуется установка)

## 🚀 Установка и запуск

### Сборка из исходников (Gradle):

```bash
# Сборка JAR
./gradlew build

# Сборка портативной версии (Windows)
.\build-portable.ps1
```

### Запуск готового JAR:

```bash
java -jar build/libs/autoservice-admin.jar
```

## 📤 Экспорт данных

### Экспорт заказа в PDF
- Откройте контекстное меню заказа
- Выберите "Сохранить заказ-наряд в PDF"
- Файл сохраняется с форматированным заказом-нарядом

### Экспорт справочников
- Через меню "Файл" → "Экспорт":
  - **Клиенты** — CSV, XML, JSON
  - **Услуги** — CSV, XML, JSON  
  - **Запчасти** — CSV, XML, JSON

### Экспорт отчёта
- Через меню "Отчёты" откройте общий отчёт
- Используйте `ReportGenerator.exportToPdf()` для сохранения PDF

## 🔧 Управление резервными копиями

Резервные копии автоматически создаются в папке `backups/`.

- Хранится до 14 последних копий
- Формат: `backup_yyyyMMdd_HHmmss.zip`
- Автоматическая очистка старых копий

## 📊 Логирование

Логи сохраняются в папке `logs/`:
- `autoservice.log` — текущий лог
- `autoservice.yyyy-MM-dd.log` — архивные логи (30 дней)

Портативная версия: `AdminSTO_Portable/logs/`
