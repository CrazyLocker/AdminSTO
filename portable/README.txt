=====================================================
  AdminSTO v1.0.0 - Портативная версия
=====================================================

ЭТА ВЕРСИЯ ДЛЯ ЗАПУСКА С ФЛЕШКИ НА ЛЮБОМ ПК С WINDOWS

=====================================================
  ОСОБЕННОСТИ
=====================================================

- ✅ Полностью автономная (не нужен JDK/JRE в системе)
- ✅ Встроенный JRE (17 MB через jlink)
- ✅ 30 библиотек (SQLite, HikariCP, Logback, iText...)
- ✅ JavaFX для Windows
- ✅ Все данные в папке portable
- ✅ Можно переносить между компьютерами

=====================================================
  СИСТЕМНЫЕ ТРЕБОВАНИЯ
=====================================================

- Windows 10 или 11 (64-bit)
- Память: 256 MB свободно
- Экран: 1024x768
- НЕ НУЖЕН JDK или JRE в системе!

=====================================================
  УСТАНОВКА
=====================================================

1. Скопируйте папку portable на флешку

2. На целевом компьютере:
   - Откройте папку portable с флешки
   - Запустите run.bat

Готово! Приложение запустится автоматически.

=====================================================
  СТРУКТУРА ПАПКИ
=====================================================

portable/
├── run.bat              - Запуск приложения
├── autoservice-admin.jar - Приложение
├── jre/                 - Встроенный JRE (17 MB)
├── lib/                 - Библиотеки (105 MB)
│   ├── javafx-*.jar     - JavaFX (4 библиотеки)
│   ├── sqlite-jdbc.jar  - База данных
│   ├── HikariCP.jar     - Пул соединений
│   ├── logback-*.jar    - Логирование
│   ├── itext7-*.jar     - PDF генерация
│   └── ... (30 библиотек)
├── logs/                - Логи приложения
├── backups/             - Резервные копии БД
└── autoservice.db       - База данных (создаётся)

=====================================================
  ХРАНЕНИЕ ДАННЫХ
=====================================================

Все данные хранятся В ПАПКЕ PORTABLE:

📁 autoservice.db     - База данных клиентов и заказов
📁 logs/              - Логи работы приложения
📁 backups/           - Резервные копии (создаются)

Это позволяет:
- ✅ Переносить данные между компьютерами
- ✅ Иметь данные всегда с собой
- ✅ Не зависеть от компьютера пользователя

=====================================================
  УПРАВЛЕНИЕ
=====================================================

Запуск:
  run.bat

Логи:
  logs/autoservice.log

Резервные копии:
  backups/backup_yyyyMMdd_HHmmss.zip
